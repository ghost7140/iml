# FREEZING POINT
## Objective:
To develop a code to check whether the given temperature is below freezing point.
## Program:
```prolog
freezing(F) :-
F =< 32.
```
## Output:
```prolog
?- freezing(15).
true

?- freezing(78).
false
```
## Result:
Thus, the code to check whether the temperature is below freezing point has been executed successfully.
